# triada-arch
Данная архитектура может содаржать неточности и использовать эту систему без проверки , не имеет смысла.
Даная архитектура построенная для  среднего бизнеса, который направлен на продажу, не на разработку!
Сеть компании
В нэтом примере будет компания с отделами: финансисты, продажники и системные администраторы
Следственно мы создаем 4 vlan'a: vlan 10(Finans), vlan 20(prod), vlan 30(sis_admin), vlan 40(dir),vlan 50(service)
Ставим ACL на каждый vlan
один сервер мы ставим для web,BD
второй сервер мы для DHCP, Active Directory(AD)
Для Web-servera испльзуйте Nginx(ubuntu server)
пример Web сайта файл- web

ПО поаоду AD, желатьельно использовать windows server, но на всякий случай напишу и на linux server
Скрипт для настрйоки AD linux server(samba) -linAD.sh
Что бы использовать откройте файл, измените данные для вашей организации,запустите файл, команды снизу
chmod +x samba-ad-setup.sh
sudo ./linAD.sh
Скрипт для настройки AD Windows server -  winAD.ps1
Что бы использовать откройте файл, измените данные для вашей организации,запустите файл
Запускать на Windows Server с ролью AD DS (от администратора)
Либо через Power Shell
Set-ExecutionPolicy RemoteSigned -Force
.\winAD.ps1
Так же для безопасности настройте SIEM систему, например ELK
